This directory contains tools and documentation for performing automated
release management:
- Manage and synchronize pluging version and POM module version
- Automated release creation

Required environment for running the release management tooling:
- Environemt Variable JAVA_HOME set to e.g. C:\Data\Software\Java\jdk1.8.0_72\jre
- On Windows only: svn command available on known path C:\SVN
  -> create link to Tortoise SVN:
     mklink /D bin "c:\Program Files\TortoiseSVN\bin"
