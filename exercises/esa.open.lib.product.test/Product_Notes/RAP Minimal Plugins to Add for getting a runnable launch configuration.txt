Steps to create minimal launch configuration for WEB-EUD

01) Create new "RAP Application" lauch configuration
02) In the Main section set the Servlet Path to /eud
03) Select the Bundles Section tab
04) Deselect all bundles
05) Add all workspace bundles EXCEPT for the eclipse dummies!!!!!
06) There shall then be about 59 selected bundles (62 with action executor)
07) Deselect "Include optional dependencies...."
08) Deselect "Add new workspace bundles ..."
09) Invoke the Add Required Bundles function
10) There shall now be 130 selected Bundles
11) add the following bundles manually:
     org.eclipse.rap.ui.forms
     org.eclipse.rap.gef
     javax.servlet
     org.eclipse.osgi.services
     org.eclipse.equinox.http.jetty
     org.eclipse.birt.report.engine.emitter.pdf
     org.eclipse.birt.report.engine.script.javascript

optionally add:
     org.eclipse.equinox.console
     org.apache.felix.gogo.shell
     
12) Invoke the Add Required Bundles function again
13) There shall be about 150 Bundles selected now.
14) Clcik "Apply" button
15) Click "Run" button
16) optionally refresh the browser window coming up if EUD is not visible immediately
